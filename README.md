# Data-Structure
高教版《算法与数据结构——C语言描述》课程笔记

## 项目简介
本笔记来源于 24 级信管郭鹏飞同学课上整理，仅供学习参考，**转载请注明出处**。

- 适用于数据结构V课程，版本参考教材为《算法与数据结构——C语言描述》（第三版），作者张乃孝等，高等教育出版社出版。
- 本笔记为**经济与管理学院**课程内容，可能与理工类专业略有不同。
- 只要有一定的 C 语言基础，能够看懂代码在做什么，基本上就可以掌握这些数据结构的原理和实现方法了

若查看过程中发现笔记中有疏漏、错误，烦请留言/更新——

---

## 阅读说明
[数据结构笔记](数据结构笔记.md)文件为复习整理笔记，内含本课程全部重点

notes 文件夹中内容对应各笔记名称，为 数据结构笔记的来源、拓展、补充。*注，标号顺序为讲课顺序，可能有一些未标号的内容编码格式未改为 UTF-8。*

[数据结构算法整理](数据结构算法整理.md)文件为历次作业题及答案汇总，可供复习参考

数据结构作业 文件夹中内容为平时课堂作业，按章节分类，内容与[数据结构算法整理](数据结构算法整理.md)相同

[数据结构程序题_押题](数据结构程序题_押题.md)文件为考前题目预测 & 考试原题分享，可供快速回顾各数据结构的构造及基本函数

[习题分析_总览](习题分析_总览_24级信管郭鹏飞.pdf)文件为清华出版社的数据结构教材课后题答案和说明，部分解题过程与高教版有所不同，可供参考
